# Visual Studio 快捷键

* ctrl + shift + space 显示参数信息
* ctrl + k, ctrl + i 显示函数信息
* ctrl + k, ctrl + c 注释
* ctrl + k, ctrl + u 取消注释
* ctrl + shift + L 删除行（不拷贝）
* ctrl + shift + V 循环粘贴
* alt + ⬆ 将代码上移
* alt + ⬇ 将代码下移
* ctrl + K, ctrl + D 格式化文档
* ctrl + K, ctrl + F 格式化所选内容
* ctrl + K, ctrl + S 用（if/try/for...）代码包围内容
* ctrl + R, ctrl + R 重命名
* ctrl + R, ctrl + O 重新排序参数
